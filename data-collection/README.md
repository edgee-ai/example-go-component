<h1 align="center">Example GoLang Data Collection Component for Edgee</h1>

This is an example of a GoLang Data Collection Edgee Component.

## Setup
Requirements:
- [edgee-cli](https://github.com/edgee-ai/edgee)
- tinygo
- wasm-tools

```shell
$ make setup
```
## Building

```shell
$ make build
```
